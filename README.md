# test_copilot

A test project to test the CoPilot agent.

## Overview

This project contains a simple Python calculator module with basic arithmetic operations, along with unit tests to validate the functionality.

## Files

- `calculator.py` – Provides `add`, `subtract`, `multiply`, and `divide` functions.
- `test_calculator.py` – Unit tests for the calculator module using `pytest`.

## Running Tests

Install pytest if needed:

```bash
pip install pytest
```

Then run the tests:

```bash
pytest test_calculator.py
```
